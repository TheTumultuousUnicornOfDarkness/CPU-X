CPU-X

CPU-X is a Free software that gathers information on CPU, motherboard and more.

Download branch Master as a .zip file Download branch Master as a tar.gz file Download the portable version (v2.2.2)

Downloads


CPU-X is similar to Windows's CPU-Z, but CPU-X is a Free and Open Source software designed for GNU/Linux; also, it works on *BSD.

It can be used in graphical mode (GTK) or in text-based mode (NCurses or dump).

CPU-X reports informations about:
  • Processor
  • Motherboard
  • Memory
  • System
  • Graphic card
  • Performance
